# Первый файл по контролю версий

Для создания подзаголовка нужно внчале поставить ##

## Выделение текста

Для выделения текста курсивом необходимо выделить текст * с двух сторон, например:

*Курсив.*

Для выделения текста полужирным необходимо выделить текст ** с двух сторон, например:

**Полужирный.**

## Списки

Для создания ненумерованных списков нужно перед элементом списка поставить *, ннапример:

* Элемент 1
* Элемент 2
* Элемент 3

Для создания нумерованных списков нужно перед элементом списка поставить 1., 2. и тд, ннапример:

1. Первый элемент нумерованного списка
2. Второй элемент нумерованного списка
